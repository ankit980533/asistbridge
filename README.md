# AssistBridge - Help Platform for Visually Impaired

A mobile and web platform designed to help visually impaired individuals request assistance from volunteers.

## Project Structure

```
assistbridge/
├── backend/          # Spring Boot API Server
├── mobile/           # Flutter Mobile App (Android + iOS)
├── admin-dashboard/  # Next.js Admin Dashboard
└── docs/             # Documentation
```

## Tech Stack

- **Mobile**: Flutter/Dart
- **Backend**: Spring Boot (Java)
- **Database**: MongoDB
- **Admin Dashboard**: Next.js + React + TailwindCSS
- **Notifications**: Firebase Cloud Messaging
- **Maps**: OpenStreetMap (flutter_map)
- **Speech**: Device Speech Recognition (speech_to_text)
- **Auth**: Firebase Authentication (OTP)
